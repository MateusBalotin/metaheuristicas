Exercício 7 — RBF para Série Temporal

σ=1 · α=1 · u={(1,0.9),(0.6,0.4)} · k=2 · θ=1 · 8 padrões · saída com 1 neurônio · UFPR MNE 740

▸ Algoritmo — RBF com pesos por mínimos quadrados
Série temporal: y = [1, 0.9, 0.75, 0.6, 0.55, 0.6, 0.4, 0.3, 0.2, 0.05]
Janela: k=2 → x = [y(t), y(t+1)], d = y(t+2)  (8 padrões)
Ativação RBF (Gaussiana):
  φⱼ(x) = exp( −||x−uⱼ||² / (2σ²) ) = exp( −||x−uⱼ||² / 2 )
Matriz G (8×3): G[i,j] = φⱼ(xᵢ),  G[i,3] = θ = 1
Solução por mínimos quadrados:
  w = (GTG)−1 GTd
Saída: ŷ = w₁·φ₁(x) + w₂·φ₂(x) + w₀·θ
• Centros fixos: u₁=(1.0, 0.9) e u₂=(0.6, 0.4)
• Pesos w calculados em batch (não online) via pseudo-inversa
• Canvas esquerdo: espaço 2D de entrada com padrões e centros
• Canvas direito (final): predições sobre a série temporal original
