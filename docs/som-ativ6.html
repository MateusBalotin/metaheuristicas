Atividade 6.1 — SOM para o Problema do Caixeiro Viajante

Burma14 (TSPLIB) · 14 cidades · anel de neurônios · topologia ring · distância Euclidiana · UFPR MNE 740

▸ Algoritmo — SOM para TSP com topologia em anel
Inicialização:
  N neurônios posicionados em elipse em torno do centróide das cidades
  wi = [cx + rx·cos(2πi/N),  cy + ry·sin(2πi/N)]
Para cada iteração e cada cidade x:
  1. Encontrar BMU:  i* = argmini d(x, wi)
      d(x, wi) = √[(x₁−wi1)² + (x₂−wi2)²]
  2. Definir vizinhança:  V = { i : dist_anel(i, i*) ≤ raio }
  3. Atualizar:  wi ← wi + α · (x − wi)  ∀ i ∈ V
Decaimento:
  αt = α₀ · 0.5t−1    raiot = raio₀ · (1 − (t−1)/(T−1))
Rota: cidades ordenadas pelo índice do BMU no anel
• N > 14 neurônios garantem que cidades diferentes possam ser mapeadas a neurônios diferentes
• Topologia em anel: dist_anel(i,j) = min(|i−j|, N−|i−j|)
• Canvas mostra mapa 2D: • cidades (azul)  · ◯ neurônios conectados (anel laranja)  · ---- rota final (verde)
