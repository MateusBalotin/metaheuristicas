Exercício 9 — Busca Tabu para o Caixeiro Viajante

10 cidades · vizinhança 2-opt · tenure k=3 · solução inicial J→D→H→B→E→A→F→I→C→G · UFPR MNE 740

▸ Algoritmo Busca Tabu
1. Criar solução inicial S₁ · i=1
Enquanto iter ≤ max_iter:
  2. Gerar vizinhança M = todos os movimentos 2-opt de S
     Para cada m: calcular f(S aplicando m)
  3. Escolher mⱼ ∈ M com menor custo tal que tabu(mⱼ)=0
     OU tabu(mⱼ)>0 mas f(Sᵢ₊₁) < f(S*) (critério de aspiração)
  4. Se f(Sᵢ₊₁) ≤ f(Sᵢ): Sᵢ = Sᵢ₊₁ · tabu(mⱼ) = k · i++
  5. Atualizar lista tabu: tabu(mₑ) = tabu(mₑ)−1 onde tabu(mₑ)>0
