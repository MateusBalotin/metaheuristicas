Exercise 4 — Adaline (WIDROW-HOFF / LMS)

22 training vectors · 3 iterations · 8 test vectors · α=0.01 · UFPR MNE 740

▸ Adaline algorithm & key differences from Perceptron
Adaline algorithm:
0. Init: w₁=0, w₂=0, θ=0, α=0.01
1. For each iteration (1…3):
  2. For each training pair (x, d):
    3. y* = θ + w₁·x₁ + w₂·x₂   ← linear activation (no δ band)
    4. e = d − y*   ← LMS error
    5. wᵢ ← wᵢ + α·e·xᵢ    θ ← θ + α·e   ← always update
    6. Classify: y = +1 if y* ≥ 0, else y = −1
  7. Check stopping: max |Δw| ≤ tolerance
⚡ Updates every pattern (not just errors) 📐 Single decision line y*=0 (no δ band) 📉 Minimises MSE = ½·e² α = 0.01 (0 < α < 1)
y*=0 (decision) 🟢 y*>0 → +1 🔴 y*<0 → −1
