Exercise 5 — MLP with Backpropagation

1 hidden layer · sigmoid activations · 22 training vectors · 8 test vectors · UFPR MNE 740

▸ Algorithm 2 — 1 hidden layer, sigmoid activations
Forward:
  z_j* = Σᵢ v_ij·xᵢ + θa_j   →   z_j = σ(z_j*) = 1/(1+e−z*)
  y* = Σⱼ w_j·z_j + θb     →   y = σ(y*)
Output layer correction:
  Δw_j = α·y·(1−y)·(d−y)·z_j   θb ← θb + α·y·(1−y)·(d−y)
Hidden layer correction:
  Δv_ij = α·(d−y)·y·(1−y)·w_j·z_j·(1−z_j)·xᵢ
  Δθa_j = α·(d−y)·y·(1−y)·w_j·z_j·(1−z_j)
Stopping: max|Δw| ≤ tolerance  or  max iterations reached
• Weights initialised randomly in [−0.5, 0.5]
• Classification: y ≥ 0.5 → class +1,  y < 0.5 → class −1
